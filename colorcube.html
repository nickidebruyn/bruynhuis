Full Description

If you enjoy brain twisting, mind-blowing endless puzzle games then this game is for you to solve and match 3 cubes in 3D of the same color.

With its unique 3 dimension color cube like grid you will be entertained for hours.
Match 3 of the same colored cubes in a horizontal line and score some serious points.
Queue the cubes so that they get squashed one row after another.

One of the unique features of Cube Match 3D is that you the player can quit the game and come back anytime later to continue playing. It will store the state of the game on exit.

Try to get on top of the leader board by score as high points as possible.
Unlock the next level of difficulty as you play.
